Linux常用命令归纳（一）

归总一些对个人而言相对有用的Linux命令，这些命令都是比较基础的。

reboot 重启
poweroff 关机
pidof [参数] [服务名称] 查询某个指定服务进程的PID号码值
kill [参数] [进程PID号] 终止某个指定PID号码的服务进程
killall [参数] [进程名称] 终止某个指定名称的服务所对应的全部进程

cd [目录名称] 切换工作路径 cd - 上次所在目录 cd ~ 当前用户的家目录 cd .. 当前目录的上级目录
pwd [选项] 显示当前所处的工作目录
ls [选项] [文件] 显示目录中的文件信息 -a参数查看全部文件（包括隐藏文件） -l参数查看文件的属性、大小等详细信息
常用 ll = ls -l ls -la la等等

cat [选项] [文件] 查看纯文本文件（较短的）-n参数显示行号
more [选项] 文件 查看纯文本文件（较长的）

mkdir [选项] 目录 创建空白的文件夹
touch [选项] [文件] 创建空白文件与设置文件的各种时间
cp [选项] 源文件 目标文件 复制文件或目录
mv [选项] 源文件 [目标路径|目标文件名]
rm [选项] 文件 删除文件或目录 -f参数强制删除 -r删除目录

tar [选项] [文件] 对文件打包压缩或解 tar -czvf 压缩包名称.tar.gz 要打包压缩的目录
tar -xzvf 压缩包名称.tar.gz 解压 -c参数压缩文件 -x参数解压文件
