Linux下安装Redis

redis下载地址
https://redis.io/download

上面有关于在Linux下安装redis的方法，这里不再重复。

设置redis自动启动：
1.设置redis.conf中daemonize为yes
vi /usr/redis/redis-3.2.4/redis.conf
2.复制redis配置文件redis.conf
mkdir /etc/redis
cp /usr/redis/redis-3.2.4/redis.conf /etc/redis/6379.conf
3.复制redis启动脚本
cp /usr/redis/redis-3.2.4//utils/redis_init_script /etc/init.d/redis
4.修改启动脚本参数，指定redis-server、redis-cli、redis.conf的副本路径
vi /etc/init.d/redis
5.设置权限
chmod 777 /etc/init.d/redis

在centos下设置开机自启动：chkconfig redisd on(unbuntu 用sudo update-rc.d redisd defaults，也可以安装rcconf，貌似没有redhat系列的chkchonfig命令好用)
