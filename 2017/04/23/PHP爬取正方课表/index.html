PHP爬取正方课表

前言

好久不写博客，没有养成当一个程序员的好习惯，羞愧万分……想想从今日后得定时写博客，对知识的积累绝对是十分有用的。
今日分享一下我在某科创项目中使用PHP爬去我们学校正方教务系统课表的心路历程。

第一步：分析表单

打开我们学校正方教务系统的登陆页面，在页面的任意处右键点击“检查”（本人使用的浏览器是Chrome），我们可以发现登陆正方教务系统的form中，要求post数据到http://202.116.160.170/default2.aspx上，而数据包括__VIEWSTATE（这个必须查看页面元素才能发现，可以通过正则表达式获取其值）、txtUserName用户名、TextBox2密码，txtSecretCode验证码，shenfen身份（吐槽一下，name的命名都不按套路啊）。

第二步：获取验证码图片

注意上图的http://202.116.160.170/CheckCode.aspx，这是我们学校正方系统验证码的链接，通过使用curl，保存当前cookie以及下载验证码图片。

第三步：模拟登陆并获取学生姓名

接着携带第一步提及的几个数据和第二步中获得的cookie，通过curl来post到http://202.116.160.170/default2.aspx上，你会发现我们已经成功模拟登陆了正方教务系统。

那为何需要获取学生姓名呢？通过上图可知，在获取课表的url上，需要传学生姓名这个参数。我们可以在模拟登陆成功后，分析页面元素，通过正则表达式将学生姓名获取下来。

第四步：获取和处理课表内容

携带之前获取的cookie以及课表url要求的学号、学生姓名和gnmkdm（常量），再次通过curl，我们可以获得整个课表页面的内容，此后利用Simple-HTML-DOM，解析课表页面的元素，对课表内容进行处理

总结

在爬虫这一块上，本人也只是使用过PHP和Python来写过一些小玩意，说一下个人体会。先说Python，有成熟的库和框架，在import大量lib的基础上，代码量少。而PHP也有不少DOM操作库，例如Simple-HTML-DOM
、 phpQuery等，如今也有拓展pthreads支持多线程操作，curl_multi并行发起请求。而两种语言在爬虫效率上的瓶颈，应当是网络IO。
