记录捣鼓ubuntu下PHP7.1下安装sqlserver拓展

由于工作需要，得使得PHP支持sqlserver拓展来操作sqlserver数据库。

微软已经提供十分详细的安装方法了https://www.microsoft.com/en-us/sql-server/developer-get-started/php/ubuntu/

但是进行到Step2.1时，通过phpinfo()查看，发现并没有安装上sqlsrv、pdo_sqlsrv拓展，至于为何没安装上，原因不明。因此google了一番，进行了下面的操作后才将拓展安装成功。

1.找到sqlsrv.so、pdo_sqlsrv.so的这两个文件的位置，我的ubuntu上这两个文件的位置分别是/usr/lib/php/20160303/sqlsrv.so、/usr/lib/php/20160303/pdo_sqlsrv.so

2.在终端输入如下命令(确保权限足够)
sudo echo “extension=/usr/lib/php/20160303/sqlsrv.so” >/etc/php/7.1/mods-available/sqlsrv.ini
sudo echo “extension=/usr/lib/php/20160303/pdo_sqlsrv.so” >/etc/php/7.1/mods-available/pdo_sqlsrv.ini
sudo ln -s /etc/php/7.1/mods-available/sqlsrv.ini /etc/php/7.1/fpm/conf.d/sqlsrv.ini
sudo ln -s /etc/php/7.1/mods-available/pdo_sqlsrv.ini /etc/php/7.1/fpm/conf.d/pdo_sqlsrv.ini
sudo service php7.1-fpm restart # 重启php-fpm

3.通过phpinfo()查看拓展是否已经安装好
